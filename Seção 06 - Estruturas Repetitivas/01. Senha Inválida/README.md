# Exercício - Par ou Ímpar

Fazer um programa para ler um número inteiro e dizer se este número é par ou ímpar.

### Exemplo de Entrada:

```
12
```

### Exemplo de Saída:

```
PAR
```

### Exemplo de Entrada:

```
-27
```

### Exemplo de Saída:

```
IMPAR
```

### Exemplo de Entrada:

```
0
```

### Exemplo de Saída:

```
PAR
```
