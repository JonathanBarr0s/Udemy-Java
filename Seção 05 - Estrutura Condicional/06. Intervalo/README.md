# Exercício - intervalo

Você deve fazer um programa que leia um valor qualquer e apresente uma mensagem dizendo em qual dos seguintes intervalos ([0,25], (25,50], (50,75], (75,100]) este valor se encontra. Obviamente se o valor não estiver em
nenhum destes intervalos, deverá ser impressa a mensagem “Fora de intervalo”.

### Exemplo de Entrada:

```
25.01
```

### Exemplo de Saída:

```
Intervalo (25,50]
```

### Exemplo de Entrada:

```
25.00
```

### Exemplo de Saída:

```
Intervalo [0,25]
```

### Exemplo de Entrada:

```
100.00
```

### Exemplo de Saída:

```
Intervalo (75,100]
```

### Exemplo de Entrada:

```
-25.02
```

### Exemplo de Saída:

```
Fora de intervalo
```
